# OFY Generated Visual Assets

Generated with the built-in image generation workflow for the OFY website revamp.

- `ofy-hero-ink-field.png`: wide red/black background field for simple hero or section backdrops.
- `ofy-diagonal-ink-mask.png`: black/white luminance mask for wipe and reveal animations.
- `ofy-prop-sheet-keyed.png`: original chroma-key prop sheet source.
- `ofy-prop-sheet.png`: transparent prop sheet after chroma-key removal.

Use `mf-mask-diagonal-ink` for CSS mask experiments and keep these assets behind content so the page remains simple.
