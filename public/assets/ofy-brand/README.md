# OddFromYou Visual Assets

Built directly from the brand assets in `brand/` and `brand/letters/` (the actual short two-O star-eyes logo, individual letter-Os with stars, and the 36 organic icon SVGs).

## Core Brand Elements Used
- Short logo: exact two star-in-oval eyes (ofy-short-logo.svg and variants)
- Single star-O from letters/letter-o-*.svg
- Organic shapes and eye/node forms from brand/*.svg (e.g. vision and monitoring icons)
- Red (#FF0000) and black versions
- Full wordmark support from smoothed source

## Visual Language (from codebase mockups + design system)
- Brutalist: solid vibrant red + flat black, high contrast
- Organic liquid/blob framing inspired by the chunky, irregular brand icon shapes
- Large negative space for HTML text / logo overlays
- Sticker / cutout energy
- No rice paper, no generic watercolor, no ink wash
