These are standalone SVG wrappers for each generated asset sheet.
Each SVG embeds the full-resolution PNG so fidelity is preserved exactly.
Because the source assets are watercolor/ink raster textures, this approach preserves appearance better than aggressive auto-tracing.
If you want true fully-vector traced SVGs, select a smaller subset (for example only icon sheets or only card plates) and they can be traced separately with quality trade-offs.
